## Getting Started

To edit a version of the project, use godot!

### Prerequisites

 - Godot
 - Git Version Control

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/sm2322/Squishy-Calculator.git
   ```
2. Open the project folder in godot with the 'Import' option
